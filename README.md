# Audio Generation Suite (Music + Vocals + SFX)

Настольное приложение на Python 3.10.11 с GUI (PySide6) для генерации:
- инструментальной музыки по текстовому описанию;
- полноценных музыкальных треков с вокалом;
- звуковых эффектов (SFX, атмосферные шумы и т.п.).

В основе:
- MusicGen (AudioCraft) для фонограмм;
- NNSVS для синтеза вокала.

## 1. Окружение

Рекомендуемая конфигурация:
- **OS**: Windows 10/11 x64.
- **Python**: 3.10.11.
- **GPU**: NVIDIA RTX 3050 Ti (4 ГБ VRAM) или аналогичная.

### Шаг 1. Создать виртуальное окружение

```bash
python -m venv .venv
.venv\Scripts\activate
```

Убедитесь, что используется Python 3.10.11.

### Шаг 2. Установить PyTorch с CUDA (под вашу GPU)

Пример для CUDA 11.8 (подходит для большинства RTX, включая 3050 Ti):

```bash
pip install torch==2.1.2+cu118 torchaudio==2.1.2+cu118 --index-url https://download.pytorch.org/whl/cu118
```

Если CUDA не установлена или возникают проблемы с драйверами, можно поставить CPU‑версию:

```bash
pip install torch==2.1.2+cpu torchaudio==2.1.2+cpu --index-url https://download.pytorch.org/whl/cpu
```

### Шаг 3. Установить зависимости проекта

```bash
pip install -r requirements.txt
```

Также потребуется установленный **FFmpeg** и наличие его в `PATH`
для корректной работы экспорта MP3/FLAC и некоторых функций `audiocraft`:

- скачать сборку FFmpeg для Windows;
- добавить путь к `bin` в переменную окружения `PATH`.

## 2. Установка и подготовка MusicGen (AudioCraft)

По умолчанию используется пакет `audiocraft` (MusicGen).
При первой генерации модель будет загружена автоматически.

Рекомендуется использовать меньшую модель (например, `small`), чтобы
уложиться в 4 ГБ VRAM:

- максимальная длительность единичной генерации: 30–60 секунд;
- при нехватке VRAM переключаться на CPU или уменьшать длительность.

## 3. Установка и подготовка NNSVS

Библиотека NNSVS устанавливается через `pip`:

```bash
pip install nnsvs
```

Для полноценного синтеза вокала необходимы:
- предобученные модели голоса (акустическая, F0, длительности и т.п.);
- конфигурационные файлы рецепта (recipe) NNSVS.

В рамках данного приложения `NNSVSService` ожидает, что:
- в конфигурации будут прописаны пути к предобученным моделям;
- установлен PyTorch с поддержкой CUDA (по желанию).

Подробности по подготовке моделей см. в официальной документации NNSVS:
`https://nnsvs.github.io/installation.html` и раздел recipes.

## 4. Быстрый тест из CLI

После установки зависимостей (и при наличии весов моделей):

```bash
python -m app.cli.generate_instrumental --prompt "epic orchestral trailer, dark and tense" --duration 15
```

и (для вокала, при настроенном NNSVS):

```bash
python -m app.cli.generate_vocal --lyrics "la la la" --voice-id default
```

Эти команды будут добавлены по мере реализации CLI‑скриптов.

## 5. Запуск GUI‑приложения

После реализации GUI:

```bash
python -m app.main
```

Будет открыто окно с вкладками:
- Музыка (инструментал);
- Песня (музыка + вокал);
- SFX / атмосферы;
- Проект (версии треков, предпрослушивание, экспорт).
